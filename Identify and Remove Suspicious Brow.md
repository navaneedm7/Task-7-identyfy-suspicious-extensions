##### &nbsp;                      **Identify and Remove Suspicious Browser Extensions**







###### **Tools Used**



Google Chrome





###### **Step 1: Access Extension Manager**



Open Chrome

Navigate to: chrome://extensions/







###### **Step 2: Review Installed Extensions**



Checked the following:



* Extension name
* Developer
* Permissions requested
* User reviews
* Installation source
* Last updated date







###### **Step 3: Extensions Identified**



|         **Extension Name**|             **Permissions**|      **Suspicious Indicators**|         **Action Taken**|
|-|-|-|-|
|          Results Boost|Read and change all website data| Not trusted by enhanced safe browsing|           Removed|
|   Cristiano Ronaldo live wallpaper|Replace the page you see when opening a new tab|       Legitimate publisher|            Kept|
|       Google Docs Offline|  Read and change few website data|       Legitimate publisher|            Kept|

###### 





**Step 4: Indicators of Suspicious Extensions**



* Requests “Read and change all your data on all websites”
* Changes default search engine or homepage
* Redirects search results
* Unknown publisher
* Very few reviews or poor ratings
* Recently installed without user knowledge
* Excessive ads/pop-ups







###### **Step 5: Removal Process**



1. Click Remove in extension manager.
2. Confirm removal.
3. Restart browser.
4. Cleared browsing data (cache \& cookies).
5. Monitored browser performance.







###### **Step 6: Post-Removal Observations**



* Improved browser speed
* Reduced pop-up ads
* No unauthorized redirects
* Stable homepage and search engine







###### **Step 7: Risks of Malicious Extensions**



Malicious browser extensions can:



* Steal login credentials
* Capture keystrokes
* Inject ads (adware)
* Redirect traffic to phishing websites
* Track browsing activity
* Install additional malware







###### **Conclusion**



During the investigation, suspicious or unnecessary extensions were identified and removed.

Browser security posture improved after cleanup. 

Regular auditing of extensions is recommended as part of endpoint security hygiene
